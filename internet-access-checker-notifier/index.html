Internet Access Checker and Notifier

Details, and a note about data rates.

When your device has poor connectivity, it might have a successful ping and say it's connected to the Internet. But that's different from being able to download enough data to do something useful. On the other end of the spectrum are speed test apps, which take forever and often fail when connectivity is available but spotty. This page is a middle ground between the two: it downloads small files then larger files which provides a test of whether there's any meaningful connectivity at all, and whether that connectivity can support more data-intensive connections. So if the page can download small files but not big files your connection might support simple messaging and text-based web pages, but not video or other high-bandwidth uses.

The data rates shown below are the average rate, not the peak rate. So they include the time required to start the connection and get it up to speed.
